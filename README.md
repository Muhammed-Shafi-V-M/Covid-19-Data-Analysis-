<h3>🧠 Project Overview</h3>
<p>
This project focuses on analyzing the impact of COVID-19 across India using state-wise and zone-wise data. 
The goal was to transform raw datasets into meaningful insights through an interactive Excel dashboard, enabling better understanding of trends, performance metrics, and regional impact.
</p>

<hr/>

<h3>🎯 Objectives</h3>
<ul>
  <li>Analyze total COVID-19 cases across Indian states and zones</li>
  <li>Compare case trends between different years</li>
  <li>Evaluate recovery, active, and death ratios</li>
  <li>Identify high and low impact regions</li>
  <li>Develop an interactive dashboard for quick insights</li>
</ul>

<hr/>

<h3>🗂️ Dataset Description</h3>
<p>
The dataset includes COVID-19 statistics for Indian States and Union Territories with key attributes such as:
</p>
<ul>
  <li>State/UTs and Zone classification</li>
  <li>Total, Active, Discharged, and Death cases</li>
  <li>Year-wise case comparison</li>
  <li>Recovery and Death Ratios</li>
  <li>Population data for comparative analysis</li>
</ul>

<hr/>

<h3>🛠️ Tools & Techniques</h3>
<ul>
  <li>Microsoft Excel</li>
  <li>Pivot Tables & Pivot Charts</li>
  <li>Slicers for interactivity</li>
  <li>KPI Cards for performance tracking</li>
  <li>Conditional Formatting for visual insights</li>
</ul>

<hr/>

<h3>📊 Dashboard Features</h3>
<ul>
  <li>📌 KPI indicators for total and yearly cases</li>
  <li>🌍 Zone-wise distribution of COVID-19 cases</li>
  <li>📈 State-wise comparison charts</li>
  <li>🏆 Top affected states analysis</li>
  <li>⚖️ Ratio analysis (Active, Recovery, Death)</li>
  <li>👥 Population vs Cases comparison</li>
  <li>🎛️ Interactive slicers for dynamic filtering</li>
</ul>

<hr/>

<h3>📈 Key Insights</h3>
<ul>
  <li>West zone showed the highest contribution to total cases</li>
  <li>Maharashtra recorded the highest number of cases</li>
  <li>Smaller regions reported significantly lower impact</li>
  <li>Recovery and death ratios varied across states</li>
  <li>Population influenced total cases but not severity alone</li>
</ul>

<hr/>

<h3>🚀 Impact</h3>
<p>
The dashboard enables quick identification of high-risk areas, supports comparative analysis, and enhances data-driven decision-making through clear and interactive visualizations.
</p>
